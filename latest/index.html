Introduction

TableWidgets

TableWidgets provides a simple set of widgets to be used in combination with Interact and JuliaDB, JuliaDBMeta (though in future other table types may be supported).

Getting started

To install it:

Pkg.clone("https://github.com/piever/TableWidgets.jl.git")
Pkg.checkout("InteractBase")
Pkg.checkout("Widgets")

See the API reference for more details.
